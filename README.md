# notifications-puge

安装
```
npm i -save notifications-puge
或
yarn add notifications-puge
```

## 参数

| 参数        | 含义         | 类型  | 是否必须  |
| ----------- |:-------------:| -----:| -----:|
|value| 显示的数值 | Number | true |
|animation| 显示的动画| String | false |
|color| 汽泡文字颜色| String | false |
|background| 汽泡背景色| String | false |
